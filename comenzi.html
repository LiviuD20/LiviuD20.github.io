Ghid Rapid de Comenzi Git

O listă cu cele mai frecvente comenzi folosite în Git Bush (terminal).

Comandă Descriere
Configurare Inițială (o singură dată)
git config --global user.name "Numele Tau" Setează numele care va apărea la 'commits'.
git config --global user.email "email@tau.com" Setează emailul care va apărea la 'commits'.
Pornirea unui Proiect
git init Inițializează un repository Git nou în folderul curent.
git clone [URL] Descarcă (clonează) un proiect existent de pe GitHub.
Lucrul Zilnic
git status Arată starea fișierelor (care sunt modificate, care sunt pregătite).
git add [fisier.txt] Adaugă un fișier specific în "staging" (pregătit pentru commit).
git add . Adaugă *toate* fișierele modificate în "staging".
git commit -m "Mesajul tău" Salvează ("fotografiază") modificările din staging în istoric.
Ramuri (Branches)
git branch Listează toate ramurile locale.
git branch [nume-ramura] Creează o ramură nouă.
git checkout [nume-ramura] Trece pe ramura specificată.
git checkout -b [nume-ramura] Creează o ramură nouă ȘI trece automat pe ea.
git merge [nume-ramura] Îmbină ramura specificată în ramura curentă.
Sincronizare cu GitHub (Remote)
git pull Descarcă modificările de pe server și le îmbină.
git push Încarcă 'commits'-urile locale pe server (GitHub).
git remote -v Arată la ce servere remote (ex: GitHub) ești conectat.
